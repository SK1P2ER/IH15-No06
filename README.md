## ğŸ“– How to Use This Project

### Option 1: View Live Sites

### Option 2: Run Locally

1. **Clone or download** this repository
   ```bash
   git clone https://github.com/YOUR-USERNAME/4-sites-4-hosts.git
   cd 4-sites-4-hosts
   ```

2. **Navigate to any site folder**
   ```bash
   cd site1-portfolio
   # or site2-weather, site3-recipes, site4-game
   ```

3. **Open in browser**
   - Simply open `index.html` in your browser
   - Or use a local server:
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js (install globally: npm install -g http-server)
   http-server -p 8000

   # PHP
   php -S localhost:8000
   ```

4. **View in browser**
   - Navigate to `http://localhost:8000`

## ğŸ† Challenges & Solutions

### Challenge 1: Cross-Platform Deployment
**Problem**: Different platforms have different configuration requirements

**Solution**: Created detailed DEPLOYMENT.md for each platform with multiple deployment methods
